Quick Start

The following instructions should get you ripping tracks in no time!

1. If you intend to rip and encode to the MP3 format, you can use lame_enc.dll (included), or you'll need to download and install the BladeEnc DLL. Download BladeEnc from here.

2. Start the program. The default location is Start->CD-DA X-Tractor->CD-DA X-Tractor.

3. Select a CD unit by pressing the Options button to bring up the CD options dialog. Place a CD into the CD drive on your computer and hit the "Refresh" button. You can cause CD-DA X-Tractor to automatically scan the TOC on startup by clicking on the Options Button on the toolbar and checking the Autoscan TOC checkbox.

4. Check your output directory and format. During the installation procedure, an "output" directory will be formed in the directory where you installed CD-DA X-Tractor. If you don't want to use this default, change it by clicking on the Options->Output Options menu item. This is also where you select the output encoder to use (Raw audio, WAV, Lame_enc, BladeEnc, Vorb_Enc).

5. Check the track(s) you want to rip. To select all tracks, click on the Edit->Select All menu item. To automatically check all tracks, check the "Autocheck track" checkbox in the Options.

6. If encoding to MP3 files, check the MP3 Options.

7. To edit the names of the tracks, either left click on the track name, and click on the Edit->Rename menu item, or right-click on the track name and select Rename from the popup menu.

8. If you have enabled ID3 tags in the MP3 options, you'll need to set the ID3 info for the CD. Click on the ID3 button to do this.

9. Click the button to begin extracting all checked tracks.

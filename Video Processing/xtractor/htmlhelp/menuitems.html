Menu Items

File
Refresh CD Rereads the TOC of the CD in the drive. Equivalent to clicking on the "Refresh" button on the toolbar.
Rip Tracks Begins ripping the tracks marked with a check mark.
Rip Segment Rips a custom segment of the disk. Used to rip just a part of a track or several tracks as if they were one.
Exit Exits the program.

Edit
Rename Renames the currently selected (hilighted in blue) track.
Select All Places a check mark next to all tracks.
Invert Selection Checks all unchecked tracks, and unchecks all checked tracks.

Options
CD Settings Brings up the CD options dialog. Used to select the CD unit to use, and jitter correction options.
Output Options Brings up the Output Options dialog. Used to select output directories and MP3/WAV output.
MP3 Settings Brings up the MP3 settings dialog. This is where to set the MP3 bitrate, and enable CRC generation and ID3 tagging.
CDDB Settings Brings up the CDDB Settings dialog. This is where to configure the CDDB server and CGI, plus HTTP proxy settings.
Edit ID3 Tag Brings up a dialog to allow editing of the ID3 tagging information.
Hicolor Buttons Selects the nicer-looking 24-bit color buttons for the toolbar

Tools
Enable ID3 An alternate place to enable ID3 tagging. If this menu item is checked, then ID3 tagging is enabled.
Enable CDDB When checked, each time the Refresh button is pressed, it will attempt to query the CDDB database. See CDDB Settings.

Help
About CD-DA X-Tractor About dialog. Lists current version of CD-DA X-Tractor, plus version of BladeEnc, if installed.
Topics Loads the HTML help files into the default browser for your system.
